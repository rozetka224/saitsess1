# CloudVault

Облачное хранилище файлов и заметок на Python с использованием Flask.

## Возможности

- **Хранение файлов**: Загрузка и скачивание файлов (изображения, документы, архивы)
- **Галерея изображений**: Просмотр загруженных изображений с превью
- **Безопасность**: Хеширование паролей, защита от несанкционированного доступа
- **Современный интерфейс**: Адаптивный дизайн, drag-and-drop загрузка

## Установка

1. Клонируйте или скачайте проект:
   ```bash
   cd oblako
   ```

2. Создайте виртуальное окружение:
   ```bash
   python -m venv .venv
   ```

3. Активируйте виртуальное окружение:
   - Windows: `.venv\Scripts\activate`
   - Linux/Mac: `source .venv/bin/activate`

4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск

1. Запустите приложение:
   ```bash
   python app.py
   ```

2. Откройте браузер и перейдите по адресу: `http://127.0.0.1:5000`

## Использование

### Регистрация

1. На главной странице нажмите «Зарегистрироваться»
2. Заполните форму: имя пользователя, email, пароль
3. После регистрации выполните вход

### Работа с файлами

1. Нажмите «Загрузить» в верхнем меню
2. Перетащите файл в зону загрузки или выберите его
3. Файлы появятся на странице «Файлы»
4. Скачивайте или удаляйте файлы по необходимости

### Работа с альбомами

1. Перейдите в раздел «Альбомы»
2. Нажмите «Перетащите файлы сюда или нажмите кнопку» для создания
3. загрузите изображения с вашего устройства
4. Нажмите «Сохранить» для сохранения

## Поддерживаемые форматы

- **Изображения**: PNG, JPG, JPEG, GIF, BMP, WebP
- **Документы**: PDF, TXT, DOC, DOCX, XLS, XLSX
- **Архивы**: ZIP, RAR, 7Z

Максимальный размер файла: 16 МБ

## Технологии

- Flask 2.x
- SQLite
- Flask-Login
- Werkzeug Security
